# Módulo 06: Métodos Baseados em Árvores e Ensembles

**Aulas 27–31 · 5 aulas · ~3,75 horas**

Árvores de decisão são a base de alguns dos algoritmos mais poderosos de ML para dados tabulares. Ensembles combinam múltiplos modelos para superar qualquer modelo individual.

## Objetivos do Módulo
- Compreender critérios de divisão (Gini, Entropia) e poda de árvores
- Implementar Random Forest e interpretar importância de features
- Aplicar XGBoost/LightGBM em competições e projetos reais
- Construir ensembles com stacking para maximizar desempenho
