# Módulo 10: Introdução ao Aprendizado Profundo

**Aulas 46–51 · 6 aulas · ~4,5 horas**

Do aprendizado profundo clássico aos modelos de linguagem de grande escala (LLMs), este módulo cobre CNNs, RNNs, Transformers e as melhores práticas modernas.

| Aula | Título |

## Objetivos do Módulo
- Implementar CNNs para classificação de imagens
- Compreender o mecanismo de atenção e a arquitetura Transformer
- Aplicar transfer learning com modelos pré-treinados
- Usar LLMs via API e HuggingFace para tarefas de NLP
