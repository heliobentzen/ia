# Módulo 05: Regressão e Classificação

**Aulas 21–26 · 6 aulas · ~4,5 horas**

Este módulo aprofunda os principais algoritmos de aprendizado supervisionado: desde a regressão linear e logística até SVM e Naive Bayes, além de técnicas de ajuste de hiperparâmetros.

## Aulas

| Aula | Título |
|------|--------|
| 21 | [Regressão Linear](aula-21-regressao-linear.md) |
| 22 | [Regressão Polinomial e Regularizada](aula-22-regressao-polinomial-regularizada.md) |
| 23 | [Regressão Logística](aula-23-regressao-logistica.md) |
| 24 | [Support Vector Machines (SVM)](aula-24-svm.md) |
| 25 | [Naive Bayes](aula-25-naive-bayes.md) |
| 26 | [Ajuste de Hiperparâmetros](aula-26-hiperparametros.md) |

## Objetivos do Módulo
- Implementar e interpretar modelos de regressão linear e logística
- Aplicar regularização Ridge, Lasso e ElasticNet
- Compreender a geometria do SVM e o papel dos kernels
- Usar GridSearchCV e Optuna para otimização de hiperparâmetros
