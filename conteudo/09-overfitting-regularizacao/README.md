**Aulas 42–45 · 4 aulas · ~3 horas**

Controlar a complexidade do modelo é fundamental para a generalização. Este módulo aprofunda as técnicas de regularização para modelos clássicos e redes neurais.

- Usar Dropout, Early Stopping e Batch Normalization
