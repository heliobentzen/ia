# Módulo 07: Redes Neurais Artificiais

**Aulas 32–36 · 5 aulas · ~3,75 horas**

Das origens biológicas ao treinamento com backpropagation, este módulo apresenta as redes neurais artificiais e sua implementação prática com Keras/TensorFlow.

## Objetivos do Módulo
- Compreender o neurônio artificial e o Perceptron de Rosenblatt
- Entender o algoritmo de backpropagation e a regra da cadeia
- Implementar MLPs completos com Keras para classificação e regressão
- Escolher adequadamente funções de ativação e otimizadores
